## 🎨 Design

「Editorial Trust」コンセプト ——
日経・東洋経済のような知的編集物の佇まいで、医療×ビジネス読者層に向けた落ち着いた紙面風デザイン。

- **Base**: `#fafaf7` (微クリーム)
- **Ink**: `#1a1a1a` (墨)
- **Accent**: `#1e3a5f` (ディープネイビー)
- **Sub**: `#b8956a` (アンバー/銅)
- **Font**: Noto Serif JP（見出し） + Noto Sans JP（本文）
